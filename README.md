## Nama Proyek

TabungIn adalah aplikasi manajemen target tabungan berbasis mobile multiplatform yang dikembangkan menggunakan framework Kotlin Multiplatform dan Compose Multiplatform dengan menerapkan pola Clean Architecture serta MVVM. Aplikasi ini memfasilitasi pengguna untuk mencatat dan memantau progres celengan fisik secara digital melalui fitur CRUD yang didukung oleh SQLDelight untuk penyimpanan lokal serta visualisasi data reaktif menggunakan State Flow. Dengan sistem navigasi antarlayar yang mendukung argument passing, TabungIn juga direncanakan mengintegrasikan Gemini API untuk asisten finansial cerdas serta fitur Dark Mode guna memberikan pengalaman pengguna yang modern, responsif, dan fungsional.
